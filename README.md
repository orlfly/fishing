钓鱼助手


# 淡水鱼类知识库

所在目录： dsyl

数据由农牧渔业部水产局、中国科学院水生生物研究所、上海自然博物馆出版的书籍：《中国淡水鱼类原色图集》 中提取

项目包含处理脚本及处理后的json格式数据。

# 海洋鱼类知识库

所在目录： hyyl

由海洋鱼类图鉴字典中提取

字典数据下载位置：https://downloads.freemdict.com/%E5%B0%9A%E6%9C%AA%E6%95%B4%E7%90%86/%E9%9B%86%E5%90%88/Animals%20and%20Plants/%E6%B5%B7%E6%B4%8B%E9%B1%BC%E7%B1%BB%E5%9B%BE%E9%89%B4/


项目包含处理脚本及处理后的json格式数据。


# 格式转换

项目提供了将json格式转换成excel 的脚本，其他格式转换自行处理。
